よくある質問

位置情報が取得できません
以下の手順をお試しください：

1. 設定 > プライバシー > 位置情報 を開く
2. 「防災ライフ」アプリを見つける
3. 「アプリの使用中は許可」または「常に許可」を選択
4. アプリを再起動してください
プッシュ通知が届きません
通知設定をご確認ください：

1. 設定 > 通知 > 防災ライフ を開く
2. 「通知を許可」をONにする
3. 「バナー」「サウンド」「バッジ」をすべてONにする
4. 「重要な通知」をONにする（iOS 15以降）
アプリが起動しません・クラッシュします
以下の方法をお試しください：

1. アプリを完全に終了し、再度起動
2. 端末を再起動
3. App Storeでアプリの最新版が利用可能か確認
4. 端末の空き容量を確認（1GB以上推奨）

それでも解決しない場合は下記フォームからお問い合わせください。
避難所の情報が表示されません
以下をご確認ください：

1. インターネット接続が安定しているか確認
2. 位置情報の許可が有効になっているか確認
3. GPS信号を受信できる場所（屋外など）で試す
4. アプリ内で地域設定が正しく行われているか確認
家族との安否確認機能の使い方を教えてください
家族安否確認機能の使い方：

1. アプリ内で家族を招待・登録
2. 緊急時は「無事です」ボタンをタップ
3. 位置情報と安否状況が自動で家族に共有されます
4. 家族の安否状況はリアルタイムで確認可能

詳細な設定方法についてはアプリ内のヘルプをご確認ください。
